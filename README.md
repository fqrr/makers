# 创客

#### 介绍
什么是创客新零售系统? 创客新零售并非电商卖货平台,创客新零售系统是一个互助升级、卖货权限、人脉资源、大数据为一身的平台,此项目是一个小程序模板，需要的小伙伴可以拿去仔细研究下
